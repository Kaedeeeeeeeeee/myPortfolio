# My Portfolio

This is my personal portfolio website, built with [Next.js](https://nextjs.org) and [Once UI](https://once-ui.com).

## Getting Started

1.  **Install dependencies**

    ```bash
    npm install
    ```

2.  **Run development server**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

-   `src/app`: Application source code and pages.
-   `src/resources`: Configuration and content files.
-   `src/components`: Reusable UI components.
